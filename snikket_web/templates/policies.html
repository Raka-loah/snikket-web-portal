{% trans %}Privacy Policy{% endtrans %}

{% trans %}What information does a Snikket service collect?{% endtrans %}

{% trans %}Basic account information{% endtrans %}

{% trans %}When you create an account on this service, your username will be stored, along with a hashed version of your password.{% endtrans %}

{% trans %}You may additionally provide a profile picture (avatar) and display name. These will be shared with other users on the network, so they are able to identify you. You can control visibility of this information in the profile section of the {{ snikket_domain }} website.{% endtrans %}

{% trans %}Contacts that you add within the app will be stored in your contact list on the {{ snikket_domain }} server. This is so that the server can identify who you have permitted to view your online status, profile and other information, and to synchronize your Snikket contacts if you have multiple apps or devices.{% endtrans %}

{% trans %}Messages{% endtrans %}

{% trans %}When you send or receive a message on Snikket, we store this temporarily in your personal “message archive” on {{ snikket_domain }}. The purpose of your message archive is to enable an app you use with your account to “catch up” on recent conversations. This allows Snikket to:{% endtrans %}

  • {% trans %}ensure delivery of messages even if you are temporarily offline or experiencing connectivity issues, and{% endtrans %}
  • {% trans %}allow synchronization of messages across multiple devices and apps that you may use.{% endtrans %}

{% trans %}The data stored for each entry in the message archive is:{% endtrans %}

  • {% trans %}A unique identifier for the message{% endtrans %}
  • {% trans %}The time and date that the message was sent/received{% endtrans %}
  • {% trans %}The sender and recipient of the message{% endtrans %}
  • {% trans %}The message contents (encrypted according to your app’s settings){% endtrans %}

{% trans %}Entries in the message archive are stored for a minimum of {{ retention_days }} days. The server will routinely erase all entries after they have been in the archive for this amount of time.{% endtrans %}

{% trans %}We encourage the use of encryption of your message contents, as is the default within the Snikket app.{% endtrans %}

{% trans %}Uploaded files{% endtrans %}

{% trans %}You may also use the server to upload files (including images and videos) within your conversations. These files will remain on the server for a minimum of {{ retention_days }} days. This allows your contacts time to retrieve the file, even if they are offline. Similarly to message archives, the server will routinely erase files beyond this age.{% endtrans %}

{% trans %}Uploaded files are assigned a long random identifier, included in the link to the file. This ensures your files can only be viewed by people you share the link with.{% endtrans %}

{% trans %}The server will store the following information for every shared file:{% endtrans %}

  • {% trans %}A unique identifier{% endtrans %}
  • {% trans %}The time and date that the file was uploaded{% endtrans %}
  • {% trans %}The file name{% endtrans %}
  • {% trans %}The file size{% endtrans %}
  • {% trans %}The file type (as reported by the app){% endtrans %}
  • {% trans %}The file contents (encrypted according to your app’s settings){% endtrans %}

{% trans %}The Snikket app will automatically encrypt file contents when sharing a file within an encrypted conversation.{% endtrans %}

{% trans %}Once you share a file with a contact, understand that the contact may store a copy of the file on their device that is beyond our control and may remain even after the file is removed from {{ snikket_domain }}.{% endtrans %}

{% trans %}Access and network information{% endtrans %}

{% trans %}The Snikket server may record the time and general location from which you connect to your account or perform certain security-related actions, such as changing your password.{% endtrans %}

{% trans %}This is to help identify unauthorized access to your account, and detect when your account becomes inactive for administrative purposes (for example, so that it may be erased when no longer needed).{% endtrans %}

{% trans %}Cookies{% endtrans %}

{% trans %}When you access your account through the {{ snikket_domain }} website, one or more small pieces of data known as “cookies” may be stored in your web browser. These essential cookies allow us to securely identify your browser as you move between different pages on {{ snikket_domain }}, and therefore protect your account from unauthorized access. The cookies are not shared with third-parties or used for tracking, advertising or any such purposes.{% endtrans %}
